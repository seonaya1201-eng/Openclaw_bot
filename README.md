# OpenClaw-101: 나만의 AI 비서에서 전문가 에이전트로

![OpenClaw School](https://img.shields.io/badge/OpenClaw-School-blue?style=for-the-badge)
![AI Agent](https://img.shields.io/badge/AI-Agent-green?style=for-the-badge)

OpenClaw School은 단순히 AI를 사용하는 것을 넘어, 자신만의 전문 지식을 갖춘 **AI 에이전트(직원)**를 직접 구축하고 상품화하는 과정을 다루는 교육 커리큘럼 저장소입니다.

## 🚀 교육 목표

- AI 에이전트의 개념을 이해하고 자신만의 대화형 비서 구축
- Obsidian을 활용한 AI 친화적인 지식 베이스(Knowledge Base) 설계
- 업무 자동화(SOP)를 통해 비서를 넘어선 'AI 직원'으로의 고도화
- 완성된 에이전트의 브랜딩 및 마켓플레이스 등록

## 🛠 주요 기술 스택

- **Platform:** [OpenClaw](https://github.com/OpenClaw) (AI 에이전트 실행 프레임워크)
- **Knowledge:** Obsidian (Markdown 기반 지식 관리)
- **Integration:** Maton API (Gmail, Calendar 등 외부 서비스 연동)
- **Channels:** Telegram, Discord (메신저 인터페이스)
